iPadPlus for iOS 6.1.x

iPadPlus ports stock iPhone apps to iPad

No screenshots for this item.

iPadPlus will install calculator, voice memos, compass, stocks and the weather and stocks widgets.

Some apps have graphical issues. You can try using FullForce to make these apps fit the screen, but there's no real solution.

This package is compatible with iOS 6.1-6.1.3
